#### STL vector  
{% highlight lua %}
std::vector<float> smt;  
std::cout << smt.size() << std::endl;  
// print 0  
smt.push_back(data);  
std::cout << smt.size() << std::endl;  
// print data  
{% endhighlight %}  
It is different from 'array'.  
The array must define its size. But, vector does not.  
So, Using vector is more comfortable in case of changing memory size.
